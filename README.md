## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

To customize the portfolio with your own information:

1. Update personal information in `src/components/Hero.jsx`
2. Modify education details in `src/components/Education.jsx`
3. Update skills in `src/components/Skills.jsx`
4. Edit experience in `src/components/Experience.jsx`
5. Add your projects in `src/components/Projects.jsx`
6. Update certifications in `src/components/Certifications.jsx`
7. Modify publications in `src/components/Publications.jsx`
8. Update contact information in `src/components/Contact.jsx`

## Color Scheme

The website uses a modern color palette defined in CSS variables:
- Primary: Indigo (#6366f1)
- Secondary: Purple (#8b5cf6)
- Text: Gray shades
- Background: White and light gray

You can customize colors in `src/index.css` by modifying the CSS variables.
